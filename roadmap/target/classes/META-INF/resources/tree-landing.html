Intro to CS

Also Refered to as Computer I

This course covers the core foundational skills often refrenced in later computer science courses. Problem solving through decomposition. Writing, debugging, and analyzing programs. Low level Algorithms for sorting and searching. Introduction to data structures, recursion.

Fundamentals of Programming

Also Refered to as Computer II

This course is devoted to various advanced topics, including the Standard Template Library, Exceptions, Recursion, Searching and Sorting, and Template Classes. Majority of the topics covered here will be used in data structures

Algorithms

Important

This courses goal is to investigate the essential properties of data structures and algorithms for operating on them; to use these structures as tools to assist algorithm design; to extend exposure to searching, sorting and hashing techniques.

Some curriculums combine Algorithms & Data Structures together

Data Structures

Important

This courses goal is to investigate the essential properties of data structures and algorithms for operating on them; to use these structures as tools to assist algorithm design; to extend exposure to searching, sorting and hashing techniques.

Computer Systems/Architecture

This course introduces the elements of computer systems from the level of basic hardware gates, through to compilers, languages and applications. Topics covered include: digital logic, memory, processors, assembly language, virtual machines, recursive descent parsing, code generation and operating systems.
